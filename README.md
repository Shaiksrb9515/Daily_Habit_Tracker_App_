# Habit Tracker App

## Setup

1. Clone the repository
2. Run `flutter pub get` to install dependencies
3. Set up Firebase for iOS and Android
4. Run the app using `flutter run`

## Features

- User Authentication
- Habit Management
- Habit Tracking
- Notifications
- Progress Overview
- Statistics
- Dark Mode
- Localization
- Social Sharing
- Data Backup

## Architecture

- State Management: Provider
- Code Organization: Clean Architecture
